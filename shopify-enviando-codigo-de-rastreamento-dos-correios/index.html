Shopify: enviando código de rastreamento dos Correios

15 Maio 2015 em tecnologia, ecommerce

Estou usando o Shopify para a loja virtual de materiais para bordado da Tati e hoje tive a necessidade de enviar às clientes o código de rastreamento dos Correios (para quando estivermos no Brasil).

Como não há uma integração pronta, personalizei o email de confirmação de pedidos de acordo com as informações deste artigo: https://docs.shopify.com/support/settings/shipping/why-is-my-shipping-provider-not-listed-when-i-fulfill-an-order

Segue o código abaixo. Espero que ajude ;)

{% if fulfillment.tracking_numbers.size > 0 %}
  <p>Aqui está o código de rastreamento:
  <b>{{ fulfillment.tracking_numbers.first }}</b>.

  {% if fulfillment.tracking_company and fulfillment.tracking_company != 'Other' %}
    <a href="{{ fulfillment.tracking_urls.first }}">Para acompanhar o andamento da entrega, clique aqui.</a>
  {% else %}
    <a href="http://websro.correios.com.br/sro_bin/txect01$.QueryList?P_LINGUA=001&P_COD_UNI={{ fulfillment.tracking_numbers.first }}">Para acompanhar o andamento da entrega, clique aqui.</a>
  {% endif %}

  </p>
{% endif %}
