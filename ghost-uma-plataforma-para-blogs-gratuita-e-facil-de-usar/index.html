Ghost: uma plataforma para blogs gratuita e fácil de usar

27 Outubro 2013 em tecnologia

Desde o lançamento desse blog venho acompanhando as notícias da plataforma Ghost e não via a hora de começar a usá-la.

A migração do Wordpress para o Ghost aconteceu há duas semanas, logo que a equipe anunciou que o download estava disponível para o público em geral.

Afinal, quais são as vantagens?

  • O objetivo é claro: publicar conteúdo; escrever; "bloggar". As pessoas começaram a usar o Wordpress pra tudo, com milhares de plugins e temas, e acabou virando um Frankenstein.

  • É rápido, muito rápido: devido à tecnologia escolhida (node.js)

  • É elegante: o tema padrão é muito agradável. Você passa menos tempo se preocupando com o visual e mais tempo escrevendo.

  • Responsivo (termo da moda): todas as funcionalidades se adaptam a qualquer dispositivo.
